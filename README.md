# 🌍 PathBloom

> **Map Your Journey. Bloom Your Story.**

PathBloom is a travel tracking platform that helps users record, organize, and visualize their travel experiences. From countries and states to smaller regions, every journey can be mapped, documented, and remembered in one place.

---

## ✨ Features

### Current

- Responsive landing page
- Interactive 3D globe
- Modern UI with smooth animations
- Login page
- Mobile-friendly navigation
- Clean and minimal design

---

## 🚀 Planned Features

- Interactive country, state, and district maps
- Mark visited places with custom colors
- Upload travel photos
- Personal travel journal
- Travel statistics and progress tracking
- Travel timeline
- User profiles
- Community sharing

---

## 🛠️ Tech Stack
